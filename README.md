# Домашнє завдання №2 — Проєктування баз даних з використанням семантичних моделей

## Мета роботи
У цьому домашньому завданні опрацьовано процес нормалізації таблиць (1НФ → 2НФ → 3НФ), побудовано ER-діаграму та створено структуру бази даних у MySQL Workbench.

## Структура репозиторію
- `normalization.md` — опис нормалізації таблиці до 1НФ, 2НФ, 3НФ  
- `schema.sql` — SQL-скрипт зі створенням таблиць  
- `screenshots/` — скріншоти за етапами виконання  
- `README.md` — загальний опис проєкту  

## Зміст завдання
1. Нормалізувати початкову таблицю до 1НФ  
2. Нормалізувати отримані таблиці до 2НФ  
3. Нормалізувати структуру до 3НФ  
4. Створити ER-діаграму  
5. Створити таблиці в MySQL на основі ER-моделі  
6. Зробити скріншот розгорнутої схеми (Workbench EER Diagram)

## Використані сутності
- clients — клієнти  
- orders — замовлення  
- products — товари  
- order_items — позиції замовлень  

## ER-діаграма
ER-діаграма сформована автоматично через Reverse Engineer.  
Скріншот знаходиться у папці `screenshots/p5_workbench_schema.png`.

## Як запустити
1. Відкрити MySQL Workbench  
2. Створити нову схему або використати наявну  
3. Виконати SQL-скрипт із файлу `schema.sql`
